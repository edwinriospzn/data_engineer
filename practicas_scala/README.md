# Prácticas de Scala con Spark

## Estructura del proyecto
- `notebooks/` - Notebooks de Jupyter con ejercicios
- `data/` - Datos de ejemplo
- `jars/` - Dependencias de Scala/Spark
- `scripts/` - Scripts de utilidad

## Cómo empezar
1. `docker-compose up -d` - Levantar el entorno
2. Acceder a `http://localhost:8888` con token `practicas123`
3. Crear un nuevo notebook Scala

## Comandos útiles
- `docker-compose logs -f` - Ver logs
- `docker-compose stop` - Detener servicios
- `docker-compose down` - Eliminar contenedores
- `docker-compose build` - Reconstruir imagen
